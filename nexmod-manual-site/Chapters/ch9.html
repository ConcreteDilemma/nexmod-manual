Chapter 9: Common Errors and How to Interpret Them

Code 13: Internal Recursion

No fix. Restart may result in duplication. Proceed only if singularity is confirmed.

Code 02: Missed Offset

Retry, but not at the same time of day. Chrono-interference may be a factor.

Code 77: Latency Overflow

Wait until motion ceases entirely. Then attempt re-entry at one-third speed.
