Chapter 5: Connecting to the Network Sub-Layer

5.1 Use Coaxial-to-Aether Adapter (sold separately)

Ensure adapter is inserted until tactile click. Signal inversion may occur if inserted twice.

5.2 Disable Firewall, Then Re-enable It

Do not interrupt this step. Re-enabling must occur before timer expiry. If unsure, wait until ambient blinking ceases.

5.3 Ping Address

Send test ping to 0.0.0.0. If looped response occurs, do not interpret as failure.
